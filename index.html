HP-Gage

HpGage.new(x:, y:, width:, height:, hp:, maxhp:, color:, bgcolor:, alpha:, bgalpha:, direction:)
変数説明デフォルト値
x:, y:ゲージを表示させる位置を指定します。0
width:ゲージの横幅を指定します。100
height:ゲージの高さを指定します。10
hp:初期値を指定します。100
maxhp:最大値を指定します。100
color:ゲージの色を指定します。C_WHITE(白)
bgcolor:ゲージの背景色を指定します。nil(指定なし)
alpha:, bgalpha:不透明度を指定します。([透明]0～255[不透明])255
direction: ゲージの動く向きを指定します。("normal" か "reverse" か "both")
"normal"は、通常でhpが少ないほど、左側に減っていきます。
"reverse"は、hpが少ないほど、右側に減っていきます。
"both"は、ゲージの両端から変動します。
具体的な動きは、サンプルをご確認ください。 		nil
表示する際に指定がなければ、"normal"として表示します。
directionで先に指定されている場合は、そちらが優先されます。

draw(direction)

ゲージを表示します。
directionを指定すると、ゲージの動く向きが変わります。
"normal"or「:normal」は、通常でhpが少ないほど、左側に減っていきます。
"reverse"or「:reverse」は、hpが少ないほど、右側に減っていきます。
"both"or「:both」は、ゲージの両端から変動します。
具体的な動きは、サンプルをご確認ください。
"normal", "reverse", "both"で、シンボルも利用可能にしました。
:normal, :reverse, :both を使えます。

Hit

ob1, ob2, bool = true
「ob1, ob2」どちらも、Spriteクラスを使います。
Spriteクラスが入っている配列でも問題ありません。

「bool」は、trueかfalseを指定します。
trueの時は、当たっているとき
falseの時は、当たっていないとき

Hit.check(), Hit.check_index(), Hit.check_with_index()
それぞれブロック処理を行いますが、ブロックにより渡す値が変化します。
なお、2つの引数が両方ともSpriteクラスの場合、値を渡すことはありません。

Hit.check()は、両方が配列になっている場合、それぞれのオブジェクトを渡します。
どちらか片方が配列になっている場合、その配列側のオブジェクトを渡します。

Hit.check_index()は、両方が配列になっている場合、それぞれの要素番号を渡します。
どちらか片方が配列になっている場合、その配列側の要素番号を渡します。

Hit.check_with_index()は、両方が配列になっている場合、それぞれの要素番号とオブジェクトを渡します。
どちらか片方が配列になっている場合、その配列側の要素番号とオブジェクトを渡します。

BGM

音楽ファイルです。

wavかmidファイルを入力してください。

play, stop

playは、再生です。

stopは、停止です。

set_volumeは、音量変更です。

sets では、DxRubyのSoundクラスの設定が行えます。
変数名.sets |bgm|
　bgm.set_volume(255)
end
などです。

title
変数名.title とするとファイル名を返します。
※拡張子は、含んでおりません。

path
変数名.path とするとファイルのパスを返します。

SetInterval

何フレームごとに処理を行うかを判断するため、秒数を入力してください。

loop

ブロックで、行いたい処理をブロックの中に記入します。

TextSelect

TextSelect.new(x:, y: text:, size:, color:, bgcolor:, bgalpha:, hover:, font_alpha:)

変数説明デフォルト値
x:, y:文字を表示させる位置を指定します。0
text:表示する文字を指定します。"sample"
size:文字サイズを指定します。20
color:文字色を指定します。C_WHITE(白)
bgcolor:背景色を指定します。C_BLACK(黒)
bgalpha:背景の不透明度を指定します。([透明]0～255[不透明])0
hover:マウスカーソルが文字に重なった時の、文字の色を指定します。nil
font_alpha:文字の不透明度を指定します。([透明]0～255[不透明])255

draw
文字を表示させます。

check()
マウスが文字に重なった時の処理を書きます。
引数で(false)を指定すると、重なってない時に実行します。

draw_check
上記の処理をまとめて行います。
この処理の利用頻度は、高いと思います。
引数で(false)を指定すると、重なってない時に実行します。

BackGround

BackGround.new(scale_x, scale_y, 画像ファイル)

scale_xの位置には、画像の横幅を指定します。１が通常サイズです。
scale_yの位置には、画像の縦幅を指定します。１が通常サイズです。
画像ファイルは、pngやjpgなど背景画像として利用するファイルを指定します。

draw
背景画像を表示します。
複数の画像を設定する場合、最後に適応した画像に応じて画面の
大きさが変わります。場合によっては、大きすぎたり小さすぎたりするため、ご注意ください。

Boolean

初期のブール値(true, false)を指定してください

デフォルト: true

chenge()

trueの時はfalseに、falseの時はtrueに切り替えます。
引数(trueかfalse)が指定されている場合、そのブール値になります。
なお、true・false以外の値が入った場合、強制的にtrueにしています。

TextBox

TextBox.new(x:, y:, width:, height:, bgcolor:, font:, interval:, controll:)

コード 説明 デフォルト値
x:, y: 表示位置を指定します。 0
width: テキストボックスの横幅を指定します 500
height: テキストボックスの縦幅を指定します 300
bgcolor: テキストボックスの色を指定します C_BLACK
font: フォントのサイズを指定します 25
interval: 文字を表示する速度を指定します。
-1を指定すると瞬間表示になります。		 5
controll: 文字を次々と表示する際に、どのコントロールを行うか指定します。
:key にすると、キーボードのEnterキーを押したときに次の文字を表示します。
:mouse にすると、マウスの左クリックを押したときに次の文字を表示します。　
:both にすると上記のどちらかを押したときに次の文字を表示します。 		:both

show(color: C_WHITE)
color: は、表示するときの文字の色を指定します。
ブロックを利用することもできます。
デフォルトは、C_WHITE(白)です。
変数.show() do |text, index|
　puts　text #現在表示されている文字を返します。
　puts　index #現在の配列にあたる要素番号を返します。
end

set()
引数には、配列を入れてください。配列に入っている文字列を探し出します。
そこにある文字を順に設定していきます。

finish
表示される文字が最後になった時の処理を入れる部分です。
ブロック形式でプログラムを入れてください。

menu_set(key:)
メニュー画面を設定するための関数です。
ここで、表示したいようにメニュー画面を作成できます。
メニューが開いている間は、テキストが進まないようになっています。
ブロック形式で設定できます。
サンプル
menu_set(key: K_ESCAPE) do
　#このような形で、好きなスタイルに設定できます。
end
なお、この「TextBox」クラス内の処理に限り、テキストの切り替えを無効化しています。
他のクラスなどの処理に関しては、無効化できませんのでご了承ください。

menu_show
menu_setで設定したメニュー画面を表示します。
できる限り、プログラムの最後の方にこのコードを入れてください。
場合によっては、他のプログラムに被り、上手く動作しなくなる可能性が
あるためです。
同じキーをもう一度押せば、メニューが閉じます。

menu_close
メニュー画面を閉じます。
フェードなどは、ありません。

save
セーブ機能です。表示されている文字の要素番号を保存します。

load
ロード機能です。セーブした要素番号まで戻します。

next
次のテキストを表示します。

pred
前のテキストを表示します。

change()
0から数えて何番目のテキストを表示するか切り替えができます。

Scene

Scene.new()

set() do; end
シーンを設定します。引数には，シンボルか文字か数字が使えます。
また、カンマ区切りで複数のシーン名を設定することもできます。
同じシーン名を設定した場合、最後に設定されたシーン設定が適用されます。
ブロック形式でシーンを設定します。

unset()
設定したシーンを無効化します。
引数にシーン名を入力した後は、呼び出しても実行されません。

call()
設定したシーンを呼び出します。
シーン設定同様、複数シーンを同時に入力できます。
ただし、入力された順になります。
